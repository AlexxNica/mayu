# Flags of the Mayu binary

Mayu provides the following command line flags. When doing `mayu -h` you
should see this.

```
Usage of mayu:
  -alsologtostderr
      log to standard error as well as files
  -cluster-directory string
      path cluster directory (default "cluster")
  -config string
      path to the configuration file (default "config.yaml")
  -log_backtrace_at value
      when logging hits line file:N, emit a stack trace (default :0)
  -log_dir string
      If non-empty, write log files in this directory
  -logtostderr
      log to standard error instead of files
  -no-git
      disable git operations
  -predef string
      tsv file with  host predefined settings
  -show-templates
      show the templates and quit
  -stderrthreshold value
      logs at or above this threshold go to stderr
  -v value
      log level for V logs
  -version
      show the version of mayu
  -vmodule value
      comma-separated list of pattern=N settings for file-filtered logging
```
